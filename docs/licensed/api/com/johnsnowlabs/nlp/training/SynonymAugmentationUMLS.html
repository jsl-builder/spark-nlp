SynonymAugmentationUMLS 

case class SynonymAugmentationUMLS(sparkSession: SparkSession, umlsMetaPath: String, codeCol: String = "code", descriptionCol: String = "description", nerCol: String = "ner", chunkCol: String = "ner_chunk") extends Product with Serializable

Contains all methods to augment any given DataFrame in terms of a Combinatorial NER Synonym Matching using UMLS. The augment function takes a DataFrame and NER PipelineModel and augments it by exactly matching Named Entities through UMLS Synonym Relation. UMLS META directory is expected to be in th FS. The DataFrame is expected to have two columns: an 'identification' (hopefully unique for user's sake) column and an 'information' column. In case the augment function is called with augmentationMode=="annotation", the 'information' column should be the output of a chunk AnnotatorType . The names of the previous columns are inputs to the augment function. The names of the Pipeline's inputs / outputs can be set freely.

Example

Augmenting a simple sentence

Define or load an NER pipeline including a chunk AnnotatorType for your source data:

val doc = new DocumentAssembler().setInputCol("text").setOutputCol("document")
val tkn = new Tokenizer().setInputCols("document").setOutputCol("token")
val embs = WordEmbeddingsModel.pretrained("embeddings_clinical", "en" , "clinical/models")
          .setInputCols("document","token").setOutputCol("embeddings")
val ner = MedicalNerModel.pretrained("ner_clinical", "en", "clinical/models")
          .setInputCols("document","token","embeddings").setOutputCol("ner")
val conv = new NerConverterInternal().setInputCols("document","token","ner").setOutputCol("ner_chunk")
val edf = ResourceHelper.spark.createDataFrame(Array(Tuple1(""))).toDF("text")
val plModel = new Pipeline().setStages(Array(doc,tkn,embs,ner,conv)).fit(edf)

Then we can create the augmenter objext and call the augment function like the following:

case class SynAugPars(input: String, pipelineModel: PipelineModel, codeCol:String, descriptionCol:String,
         language: String, doProduct: Boolean, augmentationMode: String, regexes: java.util.List[(String, String)],
         output: String)

val icd10CmPars = SynAugPars("src/test/resources/synonym-augmentation/test.csv",
    plModel,
    "id","text","ENG",true,AugmentationModes.PLAIN_TEXT,
    util.Arrays.asList(
      ("(\\s+the\\s+|unspecified\\s*|other\\s*|\\s*affected|current\\s*|\\bthe\\s*|\\bits\\s*)", ""),
      ("(\\s+in\\s+.{0,}|\\s+of\\s+.{0,}|\\s+type\\s+.{0,}|\\s+\\d+$)", "")
    ),
    "src/test/resources/synonym-augmentation/augmented.parquet"
  )
val augmenter = SynonymAugmentationUMLS(ResourceHelper.spark, "src/test/resources/synonym-augmentation/mini_umls_meta")
val syns = augmenter.augment(dta.input, dta.pipelineModel, dta.codeCol, dta.descriptionCol, dta.language,
                             dta.doProduct, dta.augmentationMode, dta.regexes).cache()
syns.orderBy("code").show(1000, false)
print(syns.count()) // Will most probably exceed the original number of unique rows dues to augmentation
